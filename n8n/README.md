# n8n workflow exports

`daily-monitor.workflow.json` (once built) is a versioned export of the live workflow in the connected n8n.cloud instance, kept here for reference, rollback, and re-import. It is not the primary authoring path -- the workflow is built and iterated on directly in n8n.cloud via the project's `n8n-mcp` MCP connection (see `.mcp.json`).
